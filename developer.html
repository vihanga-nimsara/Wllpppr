Hey — I'm Vihanga, the person behind wllpppr. I built this site because I spend way too much time staring at my desktop and wanted a wallpaper library that actually respects your attention. No clutter, no sign-ups, no tracking — just good wallpapers.

Every wallpaper here is hand-picked. Curated collection focuses on quality over quantity — wallpapers that work well across light and dark modes, with clean compositions and thoughtful color palettes.

This project is completely open source. The code is available on GitHub if you want to see how it works, contribute, or build your own version.

tech stack

Plain HTML, CSS, and JavaScript — no frameworks. Images served from Cloudinary. Backend powered by Supabase (PostgreSQL) for votes, favorites, downloads, and comments. Static hosting on Vercel. No cookies, no analytics, no tracking.

get in touch

Found a bug, want to report a wallpaper, or just want to say hi? Reach out on GitHub.
